# Freelance Contract Generator

Automatically generate freelance contracts based on project details.

## Getting Started

1. Clone this repository.
2. Install the dependencies by running `pip install -r requirements.txt` in the terminal.
3. Run the application using `gunicorn -w 1 app:app`.
4. Access the application at `http://localhost:5000`.

## Deployment

This application is designed to be deployed on Railway. Make sure to set the `PORT` environment variable.
