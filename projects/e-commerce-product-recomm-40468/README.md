
# E-commerce Product Recommendation Tool

## Installation

1. Clone the repository using `git clone`
2. Install the dependencies by running `pip install -r requirements.txt`
3. Run the application using `gunicorn web:app`

## Usage

1. Open a web browser and navigate to `http://localhost:5000`

## API Documentation

This application has a single endpoint `/` which returns a welcome message.

## Contributing

Pull requests are welcome. Please make sure to follow PEP8 and commit messages are clear.

Note: This is a very basic setup. In a real-world scenario, you would want to separate concerns into different files, use a database, and implement authentication and authorization.
