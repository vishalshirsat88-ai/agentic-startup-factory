
# AI-powered Employee Onboarding Platform

This is a simple Flask MVP for an AI-powered employee onboarding platform. It provides a basic API with two routes: `/` and `/onboarding`.

## Usage

1. Install dependencies by running `pip install -r requirements.txt`.
2. Start the application by running `gunicorn -w 1 app:app`.
3. Access the application at `http://localhost:5000`.

## Environment Variables

- `PORT`: The port number to bind the application to. Defaults to 5000.
