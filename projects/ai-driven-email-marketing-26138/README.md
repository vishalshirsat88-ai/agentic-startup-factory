
# AI-driven Email Marketing Automation

Automate email marketing campaigns using AI-powered workflows.

## Getting Started

1. Clone the repository.
2. Install the dependencies using pip: `pip install -r requirements.txt`
3. Run the application using `gunicorn`: `gunicorn app:app --bind 0.0.0.0:$PORT`
4. Access the application at http://localhost:5000

## Deployment

This application is configured for cloud deployment on Railway. Set the `PORT` environment variable to the desired port number.
