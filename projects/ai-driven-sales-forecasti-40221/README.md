
# AI-driven Sales Forecasting Tool

## Getting Started

To run the application, follow these steps:

1. Install the required dependencies by running `pip install -r requirements.txt`
2. Run the application by executing `python app.py`
3. Open a web browser and navigate to `http://localhost:5000` to see the application in action

## Running on Production

To deploy the application on a production environment, use a WSGI compliant web server such as Gunicorn. You can do this by running `gunicorn app:app` in your terminal.
